# Installation

Requirements: Python 3.8+, Linux

---

## pip (Recommended)

```bash
pip install secfesc
```

Upgrade:
```bash
pip install --upgrade secfesc
```

---

## Git (Development)

```bash
git clone https://github.com/ake13-art/secfesc.git
cd secfesc
pip install -e .
```

---

## Verify

```bash
secfetch --help
secscan --help
```

---

## Uninstall

```bash
pip uninstall secfesc
```
